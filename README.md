## Проект 1. Olympic Games EDA

### Описание

В этом проекте проводится анализ данных об Олимпийских играх.  
Основной датасет содержит информацию об участниках Олимпиад: имя спортсмена, пол, возраст, рост, вес, страну, вид спорта, год, сезон Олимпиады и полученную медаль.

Дополнительно используется таблица `region.csv`, которая позволяет связать код страны `NOC` с полным названием региона.

### Данные

Используемые файлы:

```text
olimpic.csv.zip
region.csv
```

Основные признаки датасета:

| Колонка | Описание |
|---|---|
| `ID` | идентификатор спортсмена |
| `Name` | имя спортсмена |
| `Sex` | пол |
| `Age` | возраст |
| `Height` | рост |
| `Weight` | вес |
| `Team` | команда |
| `NOC` | код страны |
| `Year` | год Олимпиады |
| `Season` | сезон: Summer или Winter |
| `Sport` | вид спорта |
| `Event` | дисциплина |
| `Medal` | медаль: Gold, Silver, Bronze или пропуск |

### Что было сделано

В ноутбуке `olympic_games_eda.ipynb` выполнены следующие шаги:

1. Загружены данные из архива `olimpic.csv.zip` и файл `region.csv`.
2. Отфильтрованы данные по зимним Олимпийским играм.
3. Таблица с участниками объединена с таблицей регионов по ключу `NOC`.
4. Посчитано количество золотых, серебряных и бронзовых медалей на зимних Олимпиадах.
5. Найдено количество чемпионов и призёров зимних Олимпиад младше 18 лет.
6. Построена столбцовая диаграмма количества мужчин, участвовавших в зимних Олимпиадах, по годам.
7. Рассчитан и визуализирован индекс массы тела `BMI` для участников зимних Олимпиад.
8. Построена зависимость роста и веса для призёров зимних Олимпиад с выделением хоккеистов.
9. Для летних Олимпийских игр за период 1990–2010 медали были перекодированы в баллы:
   - Gold = 5;
   - Silver = 2;
   - Bronze = 1;
   - без медали = 0.
10. Найдены топ-5 стран по суммарным баллам в медальном зачёте.
11. Построена сводная таблица по годам и странам из топ-5.

### Результаты

По итогам анализа для летних Олимпийских игр 1990–2010 топ-5 стран по суммарным баллам:

```text
USA
Russia
Germany
Australia
China
```

Также была построена таблица, показывающая, сколько медальных баллов каждая из этих стран набрала по годам.

## Проект 2. Crime Data EDA

### Описание

Во втором проекте проводится анализ данных о правонарушениях и остановках водителей.  
Основной акцент сделан на временном анализе правонарушений за период 2010–2013 годов и отдельном исследовании случаев `DUI` — вождения под воздействием алкоголя или других веществ.

### Данные

Используемый файл:

```text
crime.csv.zip
```

В датасете представлены данные об остановках водителей, включая дату и время, пол водителя, возраст, тип нарушения, исход остановки, факт ареста и другие признаки.

Основные признаки:

| Колонка | Описание |
|---|---|
| `id` | идентификатор записи |
| `state` | штат |
| `driver_gender` | пол водителя |
| `driver_age` | возраст водителя |
| `driver_race` | раса водителя |
| `violation` | тип правонарушения |
| `search_conducted` | проводился ли обыск |
| `contraband_found` | была ли найдена контрабанда |
| `stop_outcome` | итог остановки |
| `is_arrested` | был ли водитель арестован |

### Что было сделано

В ноутбуке `crime_data_eda.ipynb` выполнены следующие шаги:

1. Загружен датасет `crime.csv.zip`.
2. Индекс датафрейма преобразован в формат даты и времени.
3. Данные отфильтрованы за период 2010–2013 годов.
4. Построена сводная таблица количества правонарушений по полу водителя и типу нарушения.
5. Найдено помесячное количество правонарушений.
6. Построен график динамики нарушений по месяцам.
7. Найдено ежедневное количество правонарушений.
8. Рассчитано среднее количество нарушений по дням недели.
9. Отдельно отобраны случаи `DUI`.
10. Проверена зависимость количества `DUI` от дня недели.
11. Найден самый “пьяный” день недели.
12. Для самого “пьяного” дня найден час с максимальным количеством `DUI`.
13. Построены графики для подтверждения выводов.

### Результаты

В результате анализа было выявлено:

- чаще всего случаи `DUI` встречаются в субботу;
- также повышенная активность наблюдается в пятницу и воскресенье;
- самый “пьяный” час для субботы — 23:00;
- наиболее рискованный период по `DUI` в данных — поздний субботний вечер.
